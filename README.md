# Playfair Cipher

Encrypts or decrypts using the [Playfair cipher](https://en.wikipedia.org/wiki/Playfair_cipher).

## Use

    $ python3 cli.py -k mykey -m message

For usage details:

    $ python3 cli.py -h

## Features

- interactive mode
- verbose mode
- file encrypt or decrypt

## Requirements

Needs Python 3.6.

## Tests

    $ py.test
